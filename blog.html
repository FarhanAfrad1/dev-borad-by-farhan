Question-1: What are the different ways to select an element in the DOM?

Multiple ways to to select an element in the DOM.But there are 5 common ways to select an element,
  1. document.getElementsByTagName()

    2. In this way you have to pass the tag name in the parenthesis an htmlcollection will return with the matched tag name.

  2. document.getElementById()

    3. Here you have to give a specific id to the element, then pass the id to the parenthesis and it will select that specefic element

  3. document.getElementsByClassName()

    4. Same way like getElementById(), here you have to give className to the element then pass the className to the parenthesis and it will return an htmlcollection of matched element

  4. document.querySelector()

    5. Here little bit different, you can select an element the way you select the element in the css. querySelector() will give you the first matched element

  5. document.querySelectorAll()

    6. same as querySelector(), but it wll give you all the matched element as nodelist.

Question-3: What is event delegation in the DOM?

Event delegation is very simple concept. When an event happened on an element same event is found to its parent and ancestor. So instead of using event listener on the childs it is used on the parent elemens. Basically event delegation is the idea where we delegate the handling of an event to a different element instead of the actual element that recieved that event.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is when an event happened on an element first runs the handles on it then handler runs on its parent this process is go on until the last ancestor is found.It's like if there is a button inside a div which is inside a section and button is clicked but there is no handler in the button, click event will search if there is any handler on the div but there is no handler in div also then it will search there is a handler in the section and handler is asked to do a alert in the windows. it will do that. see there is not handler on the button but there is a handler on the parent of button and we didn't clicked the parent, we just clicked the button then event bubble up to its ancestor and ran the handler.This is event bubbling. If there is handler on every element then every handler would ran if we just clicked the button.

Question-5: How do you create, add, and remove elements using JavaScript?

-To create an element- I have to use document.createElement().For example I want to create a "p" element then i have to do it this way document.createElement("p"). after creating it i can store the element into a variable like this,
p = document.createElement('p')

-To add that element just i created 1st of all i have to decide or select where i want to add that element. suppose i want to add that element in the body of html then i can do like this,
document.body.appendChild(p)
this will add the element into the body of the html.

-Now to remove an element I have to select the parent of that element then i can use "removeChild()" method. suppose i want to remove that p element just i added into the body now i will do like this,
let p = document.getElementById('just-created')
document.body.removeChild('p')
here i added an id to the p element and named it 'just-created' to select that p element.
